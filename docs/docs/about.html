About

The author

Sendis.org is the personal website of Franco Michel-Sendis.

The tool

The SENDIS (Screening Evaluated Nuclear Data : Integral Survey, F. Michel-Sendis, 2017) application is a Shiny server application written in R programming language and making use, among others, of the plotly and ggplot2 graphical libraries.

The application is expected to be in continuous development phase in 2017, however, great care has already been taken to carefully cross-check the data imported from different sources.
